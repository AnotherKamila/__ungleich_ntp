cdist-type__ungleich_ntp(7)
===========================
Manage NTP client

ungleich GmbH <cdist--@--ungleich.ch>


DESCRIPTION
-----------
This cdist type (de)configures ntp client on the system.

EXAMPLES
--------

.. code-block:: sh

    __ungleich_ntp

    # Remove
    __ungleich_ntp --state absent
